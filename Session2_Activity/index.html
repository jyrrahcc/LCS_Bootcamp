Lasallian CodeSpace Bootcamp

  1. Program Overview

    The Lasallian CodeSpace Bootcamp is a comprehensive and intensive program designed to equip students with the essential skills and knowledge required for a successful career in web development. This program consists of a combination of synchronous sessions, CS50 classes, and hands-on activities, ensuring that students receive a well-rounded education in web development.

  2. Program Structure
    • Synchronous Sessions:
      • Duration: 1-2 hours per session
      • Activity: Assignments to be completed before the next synchronous session
    • CS50 Classes:
      • Duration: 2-3 hours per class
  3. Learning Outcomes

    Throughout this bootcamp, students will dive deep into various aspects of web development, covering the following topics:

    1. Introduction to web development and developer tools
      - In this initial phase, you will gain a foundational understanding of web development concepts. You'll learn about the structure of the web, the role of web developers, and essential tools used in the industry.
    2. HTML(Hypertext Markup Language)
      - HTML is the backbone of web development. You'll delve into creating the structure of web pages, understanding tags, elements, and attributes. You'll learn how to create content like text, images, links, and forms.
    3. CSS(Cascading Style Sheets)
      - CSS is used for styling web pages, making them visually appealing. You'll explore how to control the layout, fonts, colors, and design elements of your web pages, enhancing user experience.
    4. JavaScript
      - JavaScript is a powerful programming language used for adding interactivity to websites. You'll learn how to manipulate web page elements, handle events, and create dynamic web applications.
    5. Capstone Project
      - The capstone project is an opportunity to apply your knowledge and skills acquired so far to create a complete web application from scratch. This project will challenge you to use HTML, CSS, and JavaScript to build a functional and visually appealing web application.
    6. React
      - React is a popular JavaScript library for building user interfaces. You'll explore the fundamentals of React, including components, state management, and how to create interactive and reusable UI elements.
    7. Node
      - Node.js is a runtime environment that allows you to run JavaScript on the server side. You'll learn how to build server-side applications, handle HTTP requests, and create APIs using Node.js.
    8. MongoDB
      - MongoDB is a NoSQL database used for storing data in a flexible and scalable way. You'll discover how to work with databases, perform CRUD (Create, Read, Update, Delete) operations, and integrate MongoDB into your web applications.
    9. Capstone Project(Continuation)
      - Building on your earlier capstone project, you'll further develop and refine your web application, incorporating React, Node.js, and MongoDB to create a fully functional and responsive web solution.
    10. Next.js
      - Next.js is a popular React framework for building server-rendered React applications. You'll explore server-side rendering, routing, and other advanced features to enhance your web development toolkit.
    11. Capstone Project(Completion)
      - In the final phase, you will put the finishing touches on your capstone project, ensuring it meets industry standards for functionality, usability, and design. You'll have the opportunity to showcase your skills and creativity.

    Throughout the bootcamp, you will progressively build your web development expertise, from the basics of HTML and CSS to creating complex web applications and completing a comprehensive capstone project. These skills are essential for a successful career in web development, and this program provides a structured and hands-on approach to mastering them.
